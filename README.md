# Amazon Clone

A frontend clone of the Amazon homepage built using HTML and CSS.

## Features

* Amazon-inspired navigation bar
* Search bar and menu sections
* Hero section with banner image
* Product category boxes
* Footer section

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created to practice frontend web development skills and improve understanding of HTML layouts, CSS styling, and responsive design concepts.
